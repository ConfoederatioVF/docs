This is the Actions Schema for Naissance HGIS. Note that this documentation is intended for developer use.

  • config.actions.<category_key>: Object
    • <action_key>: Object
      • .name: string
      • .do_not_bind_to_feature=false: boolean - Overrides .scope behaviour for Geometry actions.
      • .draw_function*: async function(arg0_naissance_obj.ui) - The UI function that gets called when the respective button in actions palette is called.
      • .node_options: Object - Binds the given action to a node, specifying parameters to pass to .special_function as a JSON packet.
      • .scope: string[]|string - The naissance. classes to scope the Action to when palettes are brought up. 
        Note: All Geometry* scopes are automatically bound to Feature if .entities is defined.
      • .special_function: async function(arg0_json)
      • .window_options: Object
